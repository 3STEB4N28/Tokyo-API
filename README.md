# Tokyo-API
Tokyo API wrapper [Click](https://api.willz.repl.co)

# Features
- ⏱️ Easy to use
- 🚀 Super powerful: image generation and JSON responses.
- 🎉 Popular API.

# Install
```
npm i Tokyo-API
```

# Usage
```js
const mid = require('Tokyo-API')
const midKey = "You-token"

(async() => {
let data = await mid.imagesearch('amongus', midKey)
})
```
